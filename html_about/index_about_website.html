Sobre o site

Este site representa uma culminação de todo que eu sei, gosto, uma expressão de mim mesmo. Ele tem três diferentes propositos, sendo esses:

  • Um aqruivo digital de todos os meus trabalhos
  • Um projeto de desenvolvimento web
  • Uma forma de expressão artistica

Ele vai continuar sim, sendo um trabalho em contrante desenvolvimento, como forma de aprimoramento das minhas técnicas.

Performance

Até o presente momento, o site está configurado apenas para ter uma perfomance em Desktop's pretendo sim criar uma layout responsivo para dispositivos mobile.

A navegação do site é baseada no estilo indie web, sites comuns dos anos 2000, então a navegação dele e baseada em muitos casos estatica, com páginas já criadas e desenhadas sem muita alteração, mas isso não tira o fato de terem componentes dinâmicos. Como posts de Blog.

Qualquer mudança presente, será alertada na aba de Changelogs.

Curiosidade:

O Website foi pensado para imitar uma mesa de projetos, em específico, uma mesa para projetos de desenhos e criação/conserto de brinquedos, utilizando um fundo que imita o papel de blueprints. >>Documentação do Site
