Sobre o site

Este site é a culminação de tudo o que eu sei e gosto — uma expressão de mim mesmo.
Ele tem três propósitos principais:

  • Um arquivo digital de todos os meus trabalhos
  • Um projeto de desenvolvimento web
  • Uma forma de expressão artística

Ele vai continuar sendo um trabalho em constante desenvolvimento, funcionando como uma forma de aprimorar minhas técnicas.

Performance

Até o presente momento, o site está configurado apenas para rodar legal em desktops. Pretendo sim criar um layout responsivo para dispositivos mobile no futuro.

A navegação do site é baseada no estilo indie web, aqueles sites clássicos dos anos 2000. Por isso, a estrutura é majoritariamente estática, com páginas já desenhadas sem muita frescura, mas isso não tira o fato de existirem componentes dinâmicos, como os posts do Blog.

Qualquer mudança será avisada na aba de Changelogs .

Curiosidade:

O website foi pensado para imitar uma mesa de projetos. Especificamente, uma mesa de desenho e criação/conserto de brinquedos, utilizando um fundo que imita papel de blueprint. >> Documentação do Site

Propaganda

O site é totalmente livre de propagandas! Então não se preocupe, seus dados não serão coletados. E não, o site não possui nenhum conteúdo de IA.
